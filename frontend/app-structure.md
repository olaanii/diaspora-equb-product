# Flutter App Structure (MVP)

## Screens
- Onboarding + Fayda verification
- Wallet creation/binding
- Pool discovery (Tier 0–3)
- Pool detail + contribution schedule
- Payout streaming status
- Credit score & tier progress

## State
- Identity: verified hash, binding status
- Wallet: address, balances
- Pool: round, contribution status, payout schedule
- Reputation: score, tier eligibility

## Non-Functional
- Offline-safe notifications
- Transaction confirmations
- Accessibility and multilingual support
