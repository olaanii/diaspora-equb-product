# User Flows

## New User (Tier 0)
1. Sign up
2. Fayda e-ID verification
3. Wallet creation + binding to identity hash
4. Join Tier 0 Equb pool
5. Participate in contributions and streamed payout

## Proven User (Tier Upgrade)
1. Complete Tier 0 without defaults
2. Auto-eligibility for Tier 1+ pools
3. Reduced collateral requirement
4. Access larger pool sizes

## Default Scenario (First Beneficiary)
1. Beneficiary misses contribution
2. Contract freezes unreleased payout
3. Collateral slashed and distributed to pool
4. Credit score updated on-chain
5. User blocked from higher tiers
